# UX Redesign Prototype

PROTOTYPE - throwaway UI prototype for the Caixa Basilica redesign.

Question answered: what should the modernized Tkinter PDV experience look and feel like before writing the implementation spec?

Open `index.html` in a browser.

Variants:

- `?variant=A` - Command Center
- `?variant=B` - Dense Terminal
- `?variant=C` - Guided Workspace

The prototype uses fake data only. It is not production code and should not be wired to the real SQLite database.
