# Preservar historico em correcoes pos-venda

Correcoes pos-venda devem preservar historico em vez de sobrescrever a venda original como se o erro nunca tivesse acontecido. A tela pode oferecer uma experiencia simples de correcao, mas o sistema deve registrar o que mudou, quando mudou e quem fez a correcao, para proteger caixa, estoque e relatorios.
