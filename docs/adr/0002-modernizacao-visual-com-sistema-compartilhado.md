# Modernizacao visual com sistema compartilhado

A modernizacao visual do PDV deve ser feita por um sistema compartilhado de componentes Tkinter, nao por ajustes isolados em cada tela. Essa base deve padronizar botoes, campos, tabelas, modais, cabecalhos, alertas e estados vazios para manter todas as telas consistentes e reduzir retrabalho durante o redesign.
