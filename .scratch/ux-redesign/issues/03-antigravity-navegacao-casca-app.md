# 03 — [AntiGravity] Redesenhar navegação principal e casca do app

**What to build:** A casca principal do app com abas no topo e estrutura visual moderna para hospedar todas as telas redesenhadas.

**Blocked by:** 02 — [AntiGravity] Criar sistema visual compartilhado Tkinter.

**Status:** ready-for-agent

- [ ] As abas principais são Venda, Vendas e correções, Estoque, Importação, Relatórios e Configurações.
- [ ] A navegação mantém abas no topo, conforme ADR 0003.
- [ ] A casca aplica o sistema visual compartilhado e responde bem ao alvo 1366x768.
- [ ] A escolha de tema pode ser refletida na casca sem reiniciar a aplicação.
